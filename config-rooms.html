Config Rooms

One room. One key. Your team's files.

Create a private room, hand the key code to your team, and drop configs straight in — folders and zips included.

The key code never appears in a URL and is never stored — everything in a room is encrypted with it, so only people holding the code can read anything. Files up to 50 MB each; large files upload in encrypted chunks and can take a moment.
